# Duck-Lead-Data-Engineer-Case-Study-Bidco-Africa
This repository contains the submission for the Lead Data Engineer Case Study at Duck.

## Project Overview

The project simulates a real-world data engineering and analytics scenario for Bidco Africa, one of the largest FMCG manufacturers in East Africa.
The task focuses on transforming messy supermarket sales data into actionable insights about data quality, promotional performance, and price positioning.

All analysis was conducted in a Jupyter Notebook, supported by a lightweight FastAPI service exposing the computed KPIs as JSON.
